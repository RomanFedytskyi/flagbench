# FlagBench Dataset

2,000 resolution scenarios across four groups, generated by
`python -m flagbench.generate_scenarios` using a fixed random seed.
Every figure in the paper is reproducible from these files.

## Scenario groups

| File | Rows | Provenance | Purpose |
|------|------|-----------|---------|
| `scenarios/normal_ops.csv` | 800 | SYNTHETIC | Typical production distributions |
| `scenarios/boundary_conditions.csv` | 400 | DERIVED | Rollout 0%/100%, compliance transitions |
| `scenarios/fallback_trigger.csv` | 400 | DERIVED | No eligible version → forced fallback |
| `scenarios/adversarial.csv` | 400 | LITERATURE + SYNTHETIC | Known flag-system failure modes |

## How paper claims map to data

| Paper claim | Source file |
|-------------|-------------|
| Reference resolver accuracy = 93.6% | `results/summary_reference.json` |
| Unleash-sim accuracy = 67.2% | `results/summary_unleash_sim.json` |
| Compliance violations: reference 139, unleash-sim 731 | Both summary JSONs |
| Unleash-sim fails compliance_precedence property | `results/properties_unleash_sim.json` |
| Reference passes all 4 properties | `results/properties_reference.json` |

## Reproducing figures

```bash
python figures/plot_results.py
```

Reads `results/summary_*.json` and `results/properties_*.json`,
writes `figures/fig1_accuracy_heatmap.png` through `fig4_property_matrix.png`.
